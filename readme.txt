This plugin adds a box to the Wordpress admin dashboard that lists all pages that haven't been updated in 90 days.

== Description ==

Keep your sites content fresh.  This plugin adds a box to the Wordpress admin dashboard that lists all pages that haven't been updated in 90 days.  Every time you log in you'll be able to quickly see which pages might need review to ensure the content is still correct and everything is up to date.

== Installation ==

1. Upload `old-page-reminder` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The "Pages that Haven't Been Updated Recently"
